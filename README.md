# Mechanic Shop Tracker (Java Desktop Application)

## Overview

Mechanic Shop Tracker is a Java desktop application designed to help small automotive repair shops manage clients, vehicles, service records, and parts purchases in an organized and efficient way.

This project demonstrates object-oriented design, user input validation, data organization, and real-world problem solving through software development.

---

## Features

* Create and manage client profiles
* Register and track multiple vehicles per client
* Record detailed service history
* Log parts purchases including quantity and cost
* Structured data validation and user prompts
* Desktop GUI workflow for real-world usability

---

## How to Run the Source Code

1. Clone or download this repository
2. Import the project into Eclipse (File → Import → Existing Projects into Workspace)
3. Ensure JavaFX libraries are configured if required
4. Run the main application class
